# 社会动力学推演引擎

基于蒙特卡洛方法的社会经济模拟系统，模拟 10,000 人在 10 年内的生活轨迹。

## 项目简介

这是一个社会动力学推演引擎，使用蒙特卡洛模拟方法，基于中国城市居民统计数据、行为经济学研究和宏观经济预测，模拟大量个体在十年内的经济行为、职业发展、健康状况和生活质量变化。

## 核心特性

- **大规模模拟**：10,000 个智能体 × 10 年模拟周期
- **多维度建模**：涵盖经济、职业、健康、心理、社交等多个维度
- **动态宏观环境**：基于真实数据的宏观经济参数变化
- **智能体行为**：基于行为经济学的个体决策模型
- **结果分析**：生成详细的推演结果报告

## 模拟维度

### 个体属性
- 人口统计学：年龄、性别、教育水平
- 经济状况：收入、储蓄、负债、投资
- 健康状态：身体健康、心理健康、压力水平
- 社交网络：社会关系、婚姻状况、家庭情况
- 职业发展：行业、技能多样性、职业稳定性

### 宏观环境
- GDP 增长率
- 通货膨胀率
- 失业率
- 市场收益率
- AI 技术颠覆程度

## 快速开始

### 环境要求

- Python 3.8+
- NumPy

### 安装依赖

```bash
pip install numpy
```

### 运行模拟

```bash
python monte_carlo_sim.py
```

### 输出文件

- `simulation_results.json`：详细的模拟结果数据
- `推演结果报告.md`：人类可读的结果分析报告

## 项目结构

```
├── monte_carlo_sim.py      # 主模拟脚本
├── simulation_results.json  # 模拟结果数据
└── 推演结果报告.md          # 结果分析报告
```
